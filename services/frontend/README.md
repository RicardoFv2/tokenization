# Frontend

Web application and PWA for the RWA Tokenization Platform.

## Responsibility

- User-facing interface for wallet, marketplace, tokenization, and education
- Real-time updates via WebSocket (price feeds, trade notifications)
- Responsive design (Web + Mobile PWA)

## Technology

React 18 + TypeScript + Tailwind CSS
